# ai-inventory-manager-android
AI로 만든 비품관리 모바일 앱

## 주요 기능
- 📸 OCR 기반 비품 자동 등록
- 🧠 자동 분류 (전자기기 / 소모품 등)
- 📊 재고 소진 예측
- ☁️ 서버 연동 (FastAPI)
- 💬 AI 비품 관리 챗봇

## 기술 스택
- Android (Kotlin, Jetpack Compose)
- Room Database
- ML Kit OCR
- TensorFlow Lite (확장 예정)
- FastAPI (Python)

## 실행 방법
1. Android Studio에서 프로젝트 열기
2. `server` 폴더에서 서버 실행
3. 앱 실행

## 향후 확장
- QR / 바코드 스캔
- 관리자 웹 대시보드
- ERP 연동
